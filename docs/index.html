Install the Framework

- Manually

  • Download the framework files from GitHub. DOWNLOAD HERE
  • Unzip the framework files.
  • With your project open in Xcode, choose your Target. Under the General tab, find Embedded Binaries and then choose the + button.
  • Choose Add Other. Select ATKit.framework. When prompted, select Copy items if needed

- Using CocoaPods

The Aurvan.com ATKit for iOS is available as a CocoaPods pod. CocoaPods is an open source dependency manager for Swift and Objective-C Cocoa projects.

  • Create a Podfile and use it to install the Framework and its dependencies. 
    $ pod init
  • Edit the Podfile and add pod details as below. 
    target 'APPLICATION_TARGET_NAME' do
# Pods for APPLICATION_TARGET_NAME
	pod "com.aurvan.ATKit.iOS", "1.0"
end
  • Run the pod install command. This will install the APIs specified in the Podfile, along with any dependencies they may have. 
    $ pod install
